Code Rush - lived fast, died young, left a tired corpse

2011-01-31

I just stumbled on this amazing documentary, which carves out the living atmosphere at Netscape during the 1998-1999 years, the period in which Netscape released the source code for their browser as Open Source and rebranded into what we know today as Mozilla.

The documentary is focused on the programmer efforts behind the move and the startup nausea, which gives it a really interesting taste and makes it a great watch.

https://www.youtube.com/watch?v=u404SLJj7ig
